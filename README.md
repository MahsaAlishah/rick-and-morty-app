# 🧬 Rick and Morty Character Explorer

A modern React application for exploring characters from the Rick and Morty universe using the public Rick and Morty API.  
The project focuses on component architecture, custom hooks, and real-world state management patterns.

---

## 🚀 Features

- 👤 Browse and display list of characters
- 🔎 Search characters dynamically
- 📄 View detailed character information (status, species, location)
- 📺 Display episodes for each character
- 🔃 Sort episodes (ascending / descending by creation date)
- ❤️ Add / remove favorite characters
- 🪟 Favorites management via modal
- ⚡ Loading states for better UX
- 🚨 Error handling with toast notifications
- ♻️ Reusable components and clean structure

---

## 🧠 Tech Stack

- React (Functional Components)
- React Hooks (useState, useEffect, custom hooks)
- Axios
- React Hot Toast
- Heroicons
- Rick and Morty API
- Vanilla CSS

---

## ⚙️ Core Functionality

- Fetch character list from API
- Fetch character details by ID
- Fetch episode data dynamically based on character episodes
- Favorites system with state management
- Modal system for UI overlays
- Episode sorting (toggle ascending/descending)
